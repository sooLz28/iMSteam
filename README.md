# iMSteam

![Image Description](https://i.imgur.com/LXqYQA1.png)



Adds pirated alternatives on Steam's website. 

![](https://i.imgur.com/lTroyYX.png)

## Installation
### Chrome Extension (Recommended)
head over to this [link]([https://chrome.google.com/webstore/detail/ngkhabcnlbbbfgalblmejolbifddmoji](https://github.com/sooLz28/iMSteam/releases/tag/v1.4))

## How to Use:
Browse Steam, and click on a game's page to find out available free alternatives.

## Add a New Site:
Visit the site and search for anything.
Copy the search URL (without the query) and paste it in "Add New Site."

Example:
YouTube: https://www.youtube.com/results?search_query=
